# AimiliVPN v1.0.3 本地浏览器验收记录

> 验收日期：2026-07-19
> 验收范围：不依赖真实 VPN 出口的 Console 浏览器功能
> 验收环境：Codex 内置浏览器、本地回环测试服务、隔离的无敏感数据夹具

## 结论

桌面、平板、移动端布局以及本地离线资源、键盘焦点、表单可访问名称、状态播报和危险操作确认流程均通过。本次验收不包含 Ubuntu、systemd、TUN 或真实 VPN/代理出口，这些项目按发布规划保留给实机验收。

## 页面与交互覆盖

- 登录页：用户名、密码和安全登录按钮可识别并完成本地登录。
- 总览：统计卡、任务摘要、全局设置、备份下载和恢复入口正常显示。
- 实例管理：实例状态、节点表、启动/重启/日志/删除等入口正常显示。
- 全局节点：国家、可用性、质量、风险等级、风险分、缓存、延迟、时间和排序控件均具备可访问名称。
- 任务与质量：任务摘要和待处理队列页面正常显示。
- 日志与安全：存储、迁移、备份、恢复和日志筛选区域正常显示。
- 危险操作：启动实例会打开原生 `confirm` 对话框；取消后页面显示“已取消启动实例”，未发出变更请求。

## 无障碍与控制台检查

- 桌面页共检查 46 个按钮、链接、输入框和选择框，缺失可访问名称数量为 0。
- `operationStatus` 和 `content` 均使用 `aria-live="polite"` 播报状态。
- 键盘聚焦“退出登录”按钮时匹配 `:focus-visible`，显示 3 px 紫色轮廓和 2 px 偏移。
- 三种视口均无页面级横向溢出；宽表格只在所属卡片内部滚动。
- 浏览器控制台未记录 `warning` 或 `error`。

## 本地离线资源证明

浏览器资源清单共 9 项：1 个本地样式、1 个本地脚本、7 个本地 API 请求；字体、图片和视频资源数量均为 0。所有 URL 均以本地回环地址开头，外部资源数量为 0。静态门禁同时扫描所有前端模板、CSS 和 JavaScript，禁止远程字体、脚本、样式和图片依赖。

## 已知限制

本地夹具不会调用真实 systemd、TUN、VPNGate 或代理出口。相关安装、升级、连接、重启、回滚和卸载行为只在 Ubuntu 实机验收中确认。
