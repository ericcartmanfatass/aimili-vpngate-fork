# AimiliVPN v1.0.2 发布说明

v1.0.2 将多实例节点获取、质量查询、任务状态和 Console 管理收拢到全局业务层。
本说明记录当前已完成的代码和自动化验证范围；Linux、systemd、OpenVPN、TUN
以及真实网络环境验收仍需在发布候选版本上单独执行。

## 本版本完成

- 全局 VPNGate 调度：单任务锁、节点去重、成功快照、失败退避，以及单国家旧快照
  48 小时宽限后过期清理。
- 全局 Scamalytics：IP 去重、7 天缓存、每分钟限流、每日配额、失败退避队列和
  持久化质量指标。
- SQLite/JSON 双后端：SQLite 保存全局业务数据，JSON 保留兼容和回退能力。
- Console：全局设置、节点中心、任务与质量、日志与安全、配置/完整备份恢复。
- 完整恢复：真实写回节点、质量结果、黑名单、任务历史和质量指标；失败时自动回滚。
- 中文化：安装器、CLI、Console、登录页、连接维护、节点探测、代理/路由日志和 API
  用户错误均已补齐，稳定错误码和机器字段保持不变。
- 安全边界：API key 使用独立 secrets 文件，OpenVPN 正文、原始第三方响应、密码、令牌
  和系统资源字段不进入全局数据库或备份载荷。

## 自动化验证

- `node --test tests/frontend_dom.test.js`：3/3 通过。
- Python 编译检查、Console JavaScript 语法检查和 `install.sh` Bash 语法检查通过。
